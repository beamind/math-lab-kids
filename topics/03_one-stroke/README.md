# 一笔画擂台 · 半个篮球场

## 目标

用一道篮球场的题，讲清“一笔画”（欧拉路径）的判断方法：

```text
如图是半个篮球场的示意图。图中的实线部分，
至少需要几笔，才能不重复地一笔一笔画成？（虚线不画）
```

答案是 **4 笔**。专题分三步：先数奇点、再用四种颜色把四笔画出来、最后给出一般性结论。

## 核心思想

- 看一个图形要几笔，只看**奇点**的个数——奇点 = 引出奇数条线的点（这张图里都是 3 条线交汇的三岔口）。
- 偶点能一进一出顺势穿过；奇点总会多一条线，只能当某一笔的起点或终点。
- 一笔最多消化 2 个奇点，所以 **2k 个奇点 → 至少 k 笔**。
- 这张图有 **8 个奇点**（上方两条弧的 4 个脚、罚球区底部 2 个点、罚球线两端 2 个点）→ 8 ÷ 2 = **4 笔**。
- 奇点个数一定是偶数（握手定理：各点线数之和 = 边数 × 2 是偶数）。

## 观察点

- 孩子能否自己找出并数对 8 个奇点？
- 孩子是否理解“偶点穿过、奇点起收”这件事？
- 孩子能否说出一般规律：0 个奇点一笔回到原点、2 个奇点一笔不回原点、2k 个奇点要 k 笔？

## 建议使用方式

1. 先让孩子看原图，自己猜要几笔。
2. 第 1 段：一起在图上圈出奇点，数出 8 个，算出 4 笔。
3. 第 2 段：用“逐笔”按钮一笔一笔看，确认每一笔都从一个奇点到另一个奇点。
4. 第 3 段：用三角形、“日”字、“田”字几个小例子，复述一般性结论。

## 备注

图形是按原题画的简化示意图：外框 + 上方两条弧 + 罚球区（含罚球线和罚球圈外弧），罚球圈靠篮筐的半圆是虚线、不计入。如果把某条线的相交方式改了，奇点数会变，按同样方法重数即可。
