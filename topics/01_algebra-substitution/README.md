# 整体代换积木机

## 目标

让孩子理解：有些题不需要分别求出每个字母，而是要先观察题目真正关心的“整体”。

本专题使用这道题：

```text
已知 2a + 13b + 3c = 90，3a + 9b + c = 72，
求 (3b + c) / (a + 2b)。
```

## 核心思想

- 先看目标，而不是先硬算。
- 把 `3b + c` 看成一个蓝盒子 `X`。
- 把 `a + 2b` 看成一个红盒子 `Y`。
- 把原条件重新拼成 `X` 和 `Y` 的组合。
- 解出 `X` 和 `Y` 的关系，再回答原问题。

## 观察点

- 孩子能否主动说出题目要求的是两个整体的比值？
- 孩子是否还想分别求 `a`、`b`、`c`？
- 孩子能否解释为什么 `3X + 2Y = 90`？
- 孩子能否解释最后为什么答案是 `1`？

## 建议使用方式

1. 先让孩子自己看原题，说出“题目最后问什么”。
2. 打开网页，让孩子用蓝盒子和红盒子拼两条线索。
3. 不急着讲消元，先问：“两条线索里有什么可以变得一样？”
4. 做完后让孩子自己复述：为什么不用求出 `a`、`b`、`c`。
