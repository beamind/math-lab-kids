代数思维专题

整体代换积木机

这道题的关键不是分别算出 a、b、c，而是发现题目真正关心的两个整体。

原题

已知 2a + 13b + 3c = 90，3a + 9b + c = 72

求 3b + ca + 2b

先看目标

X = 3b + c Y = a + 2b

题目要求的就是 X / Y。

1

拼第一条线索

用 X 和 Y 拼出 2a + 13b + 3c。

目标：2a + 13b + 3c = 90
0 个 X 0 个 Y
2

拼第二条线索

用 X 和 Y 拼出 3a + 9b + c。

目标：3a + 9b + c = 72
0 个 X 0 个 Y

拼出来以后

先把两条线索都拼出来。
3

解二元一次方程组

题目已经转化成关于 X 和 Y 的方程组，接下来直接用消元法求解。

迁移练习

前 7 道可以继续用鼠标拼，把条件改写成 X、Y 的方程组；后 3 道撤掉提示，用笔独立完成。
