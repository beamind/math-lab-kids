代数思维专题

整体代换积木机

这道题的关键不是分别算出 a、b、c，而是发现题目真正关心的两个整体。

原题

已知 2a + 13b + 3c = 90，3a + 9b + c = 72

求 3b + ca + 2b

先看目标

X = a + 2b Y = 3b + c

题目要求的就是 Y / X。

1

拼第一条线索

用 X 和 Y 拼出 2a + 13b + 3c。

目标：2a + 13b + 3c = 90
0 个 X 0 个 Y
2

拼第二条线索

用 X 和 Y 拼出 3a + 9b + c。

目标：3a + 9b + c = 72
0 个 X 0 个 Y

拼出来以后

先把两条线索都拼出来。
3

解二元一次方程组

题目已经转化成关于 X 和 Y 的方程组，接下来直接用消元法求解。

迁移练习

练习 1：先拼一拼

已知 4a + 17b + 3c = 72，a + 8b + 2c = 28，求

3b + ca + 2b

设 X = a + 2b，Y = 3b + c。

目标：4a + 17b + 3c = 72

0X 0Y

目标：a + 8b + 2c = 28

0X 0Y
设 X = a + 2b，Y = 3b + c。原条件变成 4X + 3Y = 72，X + 2Y = 28。 解得 X = 12，Y = 8，所以 Y / X = 2 / 3。

练习 2：再拼一拼

已知 6a + 5b + 2c = 60，4a + 6b + 4c = 80，求

b + c2a + b

设 X = 2a + b，Y = b + c。

目标：6a + 5b + 2c = 60

0X 0Y

目标：4a + 6b + 4c = 80

0X 0Y
设 X = 2a + b，Y = b + c。原条件变成 3X + 2Y = 60，2X + 4Y = 80。 解得 X = 10，Y = 15，所以 Y / X = 3 / 2。

练习 3：用笔独立算

已知 5a + 7b + c = 63，2a + 8b + 3c = 72，求

2b + ca + b

设 X = a + b，Y = 2b + c。原条件变成 5X + Y = 63，2X + 3Y = 72。 解得 X = 9，Y = 18，所以 Y / X = 2。

练习 4：用笔独立算

已知 5a + 17b + 2c = 75，2a + 9b + 3c = 85，求

b + ca + 3b

设 X = a + 3b，Y = b + c。原条件变成 5X + 2Y = 75，2X + 3Y = 85。 解得 X = 5，Y = 25，所以 Y / X = 5。
