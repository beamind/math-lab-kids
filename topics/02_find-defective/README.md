# 找次品天平台

## 目标

让孩子理解一道经典题：

```text
10 个零件，其中 1 个是次品。次品不知轻重。
用天平至少称几次，一定能找到它，并确定它偏重还是偏轻？
```

答案是 **至少 3 次**。本专题分两步把它讲透：先用反证法证明“2 次不够”，再用决策树给出“3 次照着做就对”的方案。

## 核心思想

- 要分辨的情况有 10 × 2 = **20 种**（哪一个 ＋ 偏重还是偏轻）。
- 天平一次只有 3 种结果（左沉 / 平 / 右沉）。所以不管怎么称，称 2 次最多只有 3 × 3 = **9 种结果**。
- 一种结果只能对应一个结论。9 种结果喂不饱 20 种情况，必有两种情况撞在同一个结果上、分不开 → **2 次不可能**。
- 称 3 次有 3 × 3 × 3 = **27 种结果**，27 ≥ 20，才**有可能**一一对应；但“有可能”要靠一棵**决策树**真正兑现。
- 27 种里只有 20 种会真实发生，剩下 7 种“不会出现”，正好是 27 比 20 多出来的。

## 观察点

- 孩子是否抓住“天平一次只有 3 种结果，所以 2 次最多 9 种”这个上限？
- 孩子能否说出这道题一共有 20 种情况？
- 孩子能否用“9 < 20，必有两种撞在一起”解释为什么 2 次不行？
- 孩子能否分清“27 ≥ 20 只是有可能”和“决策树才是真做到”？
- 走决策树时，孩子能否同时读出“几号 ＋ 偏重还是偏轻”？

## 建议使用方式

1. 先让孩子读题，自己数出“一共有多少种情况”，引出 20。
2. 第一段：看那张 9 行的表，再看下面直接列出的 11 种“无处安放”的情况 → 体会 2 次为什么不行。
3. 第二段：对照完整的决策树，顺着分支一路读到底；再看那张 27 行对照表，确认每种结果对应哪种缺陷（含 7 种“不会出现”）。
4. 第三部分：做几道练习。零件数很大时不用硬算，查“3 的乘方速查表”即可。
5. 让孩子用自己的话复述：为什么至少 3 次。能讲出“一次只有 3 种结果、2 次最多 9 种、9 < 20”，就算真懂了。
