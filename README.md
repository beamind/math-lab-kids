# Math Lab Kids

给家里小朋友使用的浏览器数学实验室。项目先保持轻量：每个专题都是一个独立网页，重点是把数学思想做成能看见、能操作、能讲出来的小实验。

## 当前专题

- [整体代换积木机](topics/01_algebra-substitution/)
- [找次品天平台](topics/02_find-defective/)

## 项目原则

1. 每个专题只解决一个核心数学思想或核心误区。
2. 优先做可操作的数学实验，而不是堆题。
3. 每次学习控制在 10-15 分钟。
4. 记录错因，而不是只记录对错。
5. 孩子能讲出来，比做对更多题更重要。

## 本地打开

可以直接双击根目录的 `index.html`，也可以启动一个本地服务器：

```bash
python -m http.server 8000
```

然后打开：

```text
http://localhost:8000/
```

## 隐私

真实学习记录不要提交到仓库。`records/` 和 `data/private/` 已加入 `.gitignore`，可以在本地保存孩子的观察记录。
